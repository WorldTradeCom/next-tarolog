# Next Tarolog
Набор WEB-страниц для привлечения клиентов таролога.

## Порядок установки
1. Убедиться в доступности в вашей системе `npm`.
2. Клонировать репозиторий.
```Bash
git clone https://github.com/WorldTradeCom/next-tarolog
cd next-tarolog
```
3. Установить зависимости.
```Bash
npm install
```
4. Собрать оптимизированную версию.
```Bash
npm run build
```
5. Запустить релизную версию на требуемом порте.
```Bash
PORT={PORT} npm run start
```
